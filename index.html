markdown笔记

markdown简介

Markdown是一种纯文本格式的标记语言，只需要在标题内容前加#即可
一个#是一级标题，二个#是二级标题，以此类推。支持六级标题。
标准语法一般在#后跟个空格再写文字

More info: Writing

加粗 斜体 斜体加粗

加粗 左右分别用两个星号括起来
斜体 左右分别用一个星号括起来
斜体并加粗 左右分别用三个星号括起来

More info: Server

删除线

删除线 左右分别用两个波浪线括起来

More info: Generating

引用

在引用的文字前加>即可，引用可以嵌套，如加两个>> 三个>>>
